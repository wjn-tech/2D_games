Place a TTF/OTF font file here to enable a nicer UI font.

Recommended free fonts:
- Inter: https://rsms.me/inter/
- Noto Sans: https://www.google.com/get/noto/

Filename expected by default: `ui_font.ttf` (you can change the path in `WandEditor._apply_sci_fi_theme`).

After dropping a font, re-open the project so Godot imports it.
