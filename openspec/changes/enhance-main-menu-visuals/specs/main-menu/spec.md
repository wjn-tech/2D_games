## MODIFIED Requirements

### Requirement: Grand Visual Scale
The Main Menu buttons and title MUST appear significantly larger and more spacious to convey a sense of scale.

#### Scenario: Large Typography
-   Given the Main Menu is displayed
-   Then the "Start Game" button text size is at least 48px.
-   And the Title text size is at least 100px.
-   And there is noticeable letter spacing (tracking) in the button text.

### Requirement: Cinematic Interaction
Hovering over buttons MUST trigger a substantial visual response that feels impactful.

#### Scenario: Impactful Hover
-   Given the user hovers over a menu button
-   Then the button scales up by at least 15%.
-   And the letter spacing increases slightly (spreading out).
-   And the color shifts to a bright, glowing intensity (HDR).
