# Spec: Sci-Fi UI Theme

## MODIFIED Requirements

#### Requirement: Global Sci-Fi Aesthetic
The Wand Editor must use a "Modern Magic Industry" theme characterized by deep blue translucent backgrounds and cyan luminous accents.

#### Scenario: Visual Consistency
- When the Wand Editor is opened, all panels show a background color of `RGBA(13, 18, 26, 217)`.
- Borders for primary panels use a `2px` width with `Color(0.2, 0.8, 1.0)`.
- Interactive buttons show a cyan glow shadow (`shadow_size: 4`) when hovered.

#### Scenario: Iconography
- Labels for "施法延迟", "法力容量", and "等级" must be accompanied by relevant SVG icons.
- Icons should be tinted cyan to match the theme.
