# Implement Lineage and Breeding System

## Summary
引入一套深度模仿《方舟：生存进化》的繁育、变异与世代继承系统。重构角色属性为“野生等级+后天加点”的双层结构，支持 NPC 结婚、生子、变异遗传。当玩家死亡时，允许“灵魂转移”至后代继续游戏，从而实现家族式的长期玩法。

## Motivation
- **长期追求**：提供比单纯升级装备更深度的数值追求（培育满属性、异色后代）。
- **动态世界**：NPC 不再是静态背景，而是有生命周期、可繁衍的种群。
- **死亡惩罚与机遇**：死亡不再是从头开始，而是家族传承的转折点。

## Key Features
- **双层属性系统**：区分基因属性（可遗传）与后天培养（不可遗传）。
- **繁育与变异**：7.3% 概率发生属性变异，支持父系/母系独立变异计数。
- **生命周期**：婴儿期 -> 成长期 -> 成年期，需通过互动（喂食）进行留痕。
- **死亡继承**：玩家死亡后镜头跳转至后代，继承主控权，原装备掉落。

## Risks
- **数值膨胀**：若无上限，多代繁育后属性可能溢出。解决方案：引入变异次数上限（20次）与变异随机性。
- **复杂度**：重构 `CharacterData` 将影响几乎所有现有的 combat 和 UI 逻辑。须确保向后兼容或批量迁移。
