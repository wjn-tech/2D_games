## ADDED Requirements
### Requirement: Inventory UI Visuals (Precision Magitech)
The inventory interface SHALL use a "Precision Magitech" aesthetic, characterized by a Cyan/Silver palette, thin HUD-like grid lines, and decorative magic circuits.

#### Scenario: Opening Visualization
- **WHEN** the user opens the inventory
- **THEN** particles MUST converge from the screen edges to the center.
- **AND** the window MUST form with an elastic expansion animation.
- **AND** the background MUST feature a slow, hypnotic deep-sea/space flow shader.

#### Scenario: Closing Visualization
- **WHEN** the user closes the inventory
- **THEN** the window MUST shatter into particles and dissipate.

#### Scenario: Item Interactions
- **WHEN** hovering a slot
- **THEN** grid lines MUST brighten and the item icon MUST scale (1.1x).
- **WHEN** an item is Rare or higher quality
- **THEN** it MUST display a persistent extra glow or border effect.

#### Scenario: Typography
- **WHEN** text is displayed in the inventory
- **THEN** it MUST use a slender, modern sans-serif font.
